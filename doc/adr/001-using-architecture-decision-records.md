# ADR-001: Adoption of Architecture Decision Records (ADR) Approach

Date: 2026-04-24 | Status: Accepted

## Context

This project requires a standardized approach for documenting architectural and technical decisions.

An existing ADR in another project already defines this approach in detail.

## Decision

We adopt the ADR standard as defined in:

ADR-001: "Using Architecture Decision Records" (2026-01-29) of the [ARIUS Archive Service](https://github.com/aerius/archive-service)
